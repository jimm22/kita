# KITA | Quick Start

### Prerequisites

Make sure you have the following installed:

1. **Install pnpm**
   ```bash
   # Install pnpm using Homebrew
   brew install pnpm
   ```

2. **Install Node.js** (v18.17 or later)
   ```bash
   # Install via Homebrew
   brew install node
   ```

### Setting up

1. **Clone the repository**
   ```bash
   git clone https://github.com/jimm22/kita.git
   ```

2. **Navigate to project and install dependencies**
   ```bash
   cd kita
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
```
